Course Description

Visual Media is a course created to introduce students to the concepts and skills needed to work in the fields of media, journalism, public relations. In this course we heavly work with the adobe program, creative cloud as well as using our writing skills to create memos for each project that we are assigned.


About my Professor

Carlos Evia majored in communications as an undergraduate and went on to recieve his masters degree in computer science. These to different fields allow for him to have a different out look on visual media when it concerns things such as website design becuase he knows both how to code the website for functionality, and how to make it look visually pleasing to veiwers.
